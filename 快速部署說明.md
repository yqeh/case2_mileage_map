# 快速部署到 Vercel

## 方式一：使用自動化腳本（推薦）

### 預覽環境部署
雙擊執行：`部署到Vercel.bat`

### 生產環境部署
雙擊執行：`部署到Vercel生產環境.bat`

## 方式二：使用 Vercel Dashboard（最簡單）

1. **將專案推送到 GitHub**
   ```bash
   git add .
   git commit -m "準備部署"
   git remote add origin <你的GitHub倉庫URL>
   git push -u origin master
   ```

2. **在 Vercel Dashboard 部署**
   - 前往 https://vercel.com/dashboard
   - 點擊「Add New Project」
   - 選擇您的 GitHub 倉庫
   - 設定環境變數（見下方）
   - 點擊「Deploy」

## 方式三：使用命令列

```bash
# 登入 Vercel（首次使用）
vercel login

# 部署到預覽環境
vercel

# 部署到生產環境
vercel --prod
```

## 必須設定的環境變數

在 Vercel Dashboard 的專案設定中，新增以下環境變數：

```
SECRET_KEY=your-secret-key-here
JWT_SECRET_KEY=your-jwt-secret-key-here
DEBUG=False
DATABASE_URI=your-database-uri-here
GOOGLE_MAPS_API_KEY=your-google-maps-api-key-here
```

**重要**：
- `DATABASE_URI`：必須使用外部資料庫（不能使用 SQLite）
- `GOOGLE_MAPS_API_KEY`：必須設定有效的 Google Maps API Key

## 部署後檢查

部署完成後，測試以下 URL：
- 健康檢查：`https://your-project.vercel.app/health`
- 前端頁面：`https://your-project.vercel.app/`
- API 端點：`https://your-project.vercel.app/api/health`
